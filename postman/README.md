# Collections

As collections do Postman da API REST deste projeto moram aqui.

Elas usam os mesmos endpoints, mas, por questões de organização e ergonomia, são divididas em três:

## `sandbox`

Para desenvolvedores que querem interagir com a API na prática, em desenvolvimento local. Exporte no GUI do Postman e trate como um ambiente livre para interação.

Contém funcionalidades de autenticação e busca de recursos automática para eliminar boilerplate nos testes pontuais.

## `for-docs`

Exclusivamente para geração de documentação estática.

## `smoke-tests`

Exclusivamente para testes de CI.
